关于 Yazi

Yazi 是一个用 Rust 编写的终端文件管理器，支持异步 I/O、图像预览（Kitty/wezterm 协议）、多标签页、丰富的插件系统，以及完整的 Vim 风格键位。

环境信息：

  • Yazi v26.5.6
  • 主题：Dracula
  • 核心增强插件：augment-command

安装

# Arch / CachyOS
sudo pacman -S yazi

# 安装依赖（预览增强）
sudo pacman -S ffmpegthumbnailer p7zip jq poppler fd ripgrep fzf zoxide imagemagick

全局设置 (yazi.toml)

[mgr]
ratio = [1, 3, 4]              # 父目录:当前目录:预览 比例
sort_by = "extension"          # 按扩展名分组排序
sort_sensitive = true          # 区分大小写
sort_reverse = false           # 正序
sort_dir_first = true          # 目录优先
show_hidden = true             # 显示隐藏文件
show_symlink = true            # 显示符号链接目标
linemode = "btime"             # 行信息显示创建时间

[preview]
wrap = "yes"                   # 预览换行
image_filter = "lanczos3"      # 图片缩放算法（高质量）

[tasks]
image_alloc = 1073741824       # 图片缓存 1GB

核心特点：

  • 默认显示隐藏文件，无需按 . 切换
  • 按创建时间排序查看（可随时切换其他排序方式）
  • 布局比例偏向预览面板（右边占 50%）
  • 鼠标可直接拖拽面板分隔线调整宽度

主题 (theme.toml)

使用 Dracula 暗色主题，配置极简：

[flavor]
dark = "dracula"

Yazi 默认快捷键概览

Yazi 的默认键位设计结合了 Vim 哲学和现代文件管理器的便利性。以下是常用默认快捷键，以及通过 augment-command 增强后的差异。

默认光标移动

快捷键 功能
k 上一个文件
j 下一个文件
g g 跳转到列表顶部
G 跳转到列表底部
<C-d> 向下半页
<C-u> 向上半页

默认文件操作

快捷键 功能
Enter / l 进入目录或打开文件
h / Backspace 返回上级目录
<Space> 切换选择状态

y 复制（yank）
x 剪切
p 粘贴
d 移到回收站
D 永久删除
a 创建文件或目录

提示：Yazi 默认键位中 j = 下、k = 上（标准 Vim 布局），本配置中通过 augment-command 调整为 i = 上、k = 下（因为 j 被拿来做了返回上级）。

撤销重做： u 撤销、<C-r> 重做

augment-command：智能增强核心

augment-command 是本配置的灵魂插件，完整配置（init.lua）：

require("augment-command"):setup({
    prompt = false,
    default_item_group_for_prompt = "hovered",
    smart_enter = true,                    -- 智能进入（目录→进入，文件→打开，压缩包→解压）
    smart_paste = false,
    smart_tab_create = false,
    smart_tab_switch = false,
    confirm_on_quit = true,                -- 多标签退出时确认
    open_file_after_creation = false,
    enter_directory_after_creation = false,
    use_default_create_behaviour = false,
    enter_archives = true,                 -- 自动解压压缩包
    extract_retries = 3,                   -- 输错密码重试次数
    recursively_extract_archives = true,   -- 递归解压（压缩包套压缩包）
    preserve_file_permissions = false,
    encrypt_archives = false,
    encrypt_archive_headers = false,
    reveal_created_archive = true,         -- 创建压缩包后定位到它
    remove_archived_files = false,
    must_have_hovered_item = true,
    skip_single_subdirectory_on_enter = true,   -- 进入时跳过单子目录
    skip_single_subdirectory_on_leave = true,   -- 返回时跳过单子目录
    smooth_scrolling = false,
    wraparound_file_navigation = true,     -- 文件列表循环导航
    protected_directories = {},
})

核心增强逻辑：

  • 选中 vs 悬停：当光标所在的文件也被选中时，操作作用于所有选中文件，否则只操作当前文件
  • 空格连续选择：按空格选中文件后，光标自动移到下一行，方便连续多选
  • 批量操作：复制/删除/粘贴/shell 命令自动适配选中的多个文件
  • 自动解压：按 l / → 进入压缩包，自动解压并进入
  • 智能跳过：当目录内只有一个子目录时，自动跳过减少按键次数
  • 循环导航：文件列表到底后自动回到开头

日常使用流程

  1. 浏览文件：i / k 上下移动，l 进入目录/打开文件/解压压缩包
  2. 多选操作：空格连续选择，然后 y 复制或 d 删除或 ; 执行 shell 命令
  3. 快速跳转：g h 回家，z fzf 搜索，Z zoxide，[ / ] 书签
  4. 预览调整：L 扩大预览，鼠标拖拽分隔线，H 恢复
  5. 压缩解压：选中文件 c z 创建压缩包，光标在压缩包上按 l 自动解压
  6. 退出：单个标签按 q 直接退出，多个标签按 q 确认

Yazi 与常见编辑器的集成

在 .bashrc / config.fish 中设置别名，让你的终端编辑器默认用 Yazi 做文件浏览：

# Fish Shell
function y
    set tmp (mktemp -t "yazi-cwd.XXXXXX")
    yazi --cwd-file=$tmp
    if test -f "$tmp"
        set dir (cat $tmp)
        if test -n "$dir" -a "$dir" != "$PWD"
            cd $dir
        end
        rm -f $tmp
    end
end
bind \eo y

这样就可以在终端中按 <C-o> 快速调起 Yazi 选文件，退出时自动进入所选目录。
