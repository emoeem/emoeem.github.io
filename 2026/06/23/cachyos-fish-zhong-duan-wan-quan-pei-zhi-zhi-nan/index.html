🧠 无感集成函数详解

核心设计：用函数覆盖常用命令，不带参数时自动弹出 fzf，带参数时直接执行原功能。
你不需要记任何快捷键，像平常一样输入命令即可。

文件编辑：v / e

function v -d "用 fzf 选文件并编辑"
    if count $argv >/dev/null
        $EDITOR $argv        # 有参数 → 直接打开
    else if type -q fzf
        set file (fzf --preview 'bat --color=always --line-range :500 {}' --preview-window right:50% --prompt "Edit> ")
        if test -n "$file"
            $EDITOR "$file"   # 无参数 → fzf 选文件 → 打开
        end
    end
end
用法 效果
v 弹出 fzf 选文件 → 选中后用 $EDITOR 打开
v ~/.config/fish/config.fish 直接 nvim ~/.config/fish/config.fish，不弹 fzf

目录跳转：c / fcd

function c -d "用 fzf 选目录并 cd"
    if count $argv >/dev/null
        cd $argv              # 有参数 → 直接 cd
    else if type -q fzf; and type -q fd
        set dir (fd --type d --hidden --follow --exclude .git | fzf --preview 'eza -T --level=2 --color=always {} | head -200' --preview-window right:50% --prompt "Cd> ")
        if test -n "$dir"
            cd "$dir"          # 无参数 → fzf 选目录 → cd
        end
    end
end
用法 效果
c 弹出 fzf 选目录 → 选中后自动 cd
c ~/Downloads 直接 cd ~/Downloads，不弹 fzf

进程管理：k / fkill

function k -d "用 fzf 选进程并 kill"
    if count $argv >/dev/null
        kill $argv            # 有参数 → 直接 kill
    else if type -q fzf
        set pid (ps aux | sed 1d | fzf --multi --header "[Ctrl+A 全选 | Tab 多选 | Enter 确认]" --prompt "Kill> " | awk '{print $2}')
        if test -n "$pid"
            kill $pid          # 无参数 → fzf 选进程 → kill
        end
    end
end
用法 效果
k 弹出 fzf 进程列表（支持 Tab 多选）→ 选中后 kill
k 12345 直接 kill 12345，不弹 fzf

SSH 连接：s / fssh

function s -d "用 fzf 选 SSH 主机并连接"
    if count $argv >/dev/null
        ssh $argv             # 有参数 → 直接 ssh
    else if type -q fzf
        set hosts (cat ~/.ssh/known_hosts 2>/dev/null | awk '{print $1}' | sed 's/\[//g; s/\]//g; s/:.*//g' | sort -u)
        set config_hosts (awk '/^Host / && !/\*/ {print $2}' ~/.ssh/config 2>/dev/null)
        set all_hosts (echo $hosts $config_hosts | tr ' ' '\n' | sort -u | string collect)
        set target (echo $all_hosts | fzf --prompt "SSH> ")
        if test -n "$target"
            ssh "$target"      # 无参数 → fzf 选主机 → ssh
        end
    end
end
用法 效果
s 从 known_hosts 和 ~/.ssh/config 中选主机 → 连接
s user@192.168.1.1 直接 ssh user@192.168.1.1，不弹 fzf

Git 分支切换：gco

function gco -d "用 fzf 选 Git 分支并 checkout"
    if count $argv >/dev/null
        git checkout $argv    # 有参数 → 直接 checkout
    else if type -q fzf
        set branch (git branch -a --format='%(refname:short)' 2>/dev/null | fzf --prompt "Checkout> " --preview 'git log --oneline --color=always {1} | head -20' --preview-window right:50%)
        if test -n "$branch"
            git checkout "$branch"  # 无参数 → fzf 选分支 → checkout
        end
    end
end
用法 效果
gco 弹出所有分支（含远程），右侧预览最近 20 条提交 → 选中后 checkout
gco main 直接 git checkout main，不弹 fzf

Git 查看提交：gsh

function gsh -d "用 fzf 浏览 Git 提交并 show"
    if count $argv >/dev/null
        git show $argv        # 有参数 → 直接 show
    else if type -q fzf
        set commit (git log --oneline --all --color=always | fzf --ansi --prompt "Show> " --preview 'git show --color=always {1} | delta' --preview-window right:60% | awk '{print $1}')
        if test -n "$commit"
            git show "$commit"  # 无参数 → fzf 选提交 → show（带 delta 美化）
        end
    end
end
用法 效果
gsh 弹出提交历史，右侧 delta 美化 diff → 选中后 show
gsh HEAD~3 直接 git show HEAD~3，不弹 fzf

Pacman 包管理：pac / pacr

function pac -d "用 fzf 搜索包并安装"
    if count $argv >/dev/null
        sudo pacman -S $argv  # 有参数 → 直接安装
    else
        set pkg (pacman -Slq 2>/dev/null | fzf --multi --prompt "Install> " --preview 'pacman -Si {1} 2>/dev/null | head -30' --preview-window right:50%)
        if test -n "$pkg"
            sudo pacman -S $pkg  # 无参数 → fzf 搜仓库包 → 安装
        end
    end
end

function pacr -d "用 fzf 搜索已安装包并卸载"
    if count $argv >/dev/null
        sudo pacman -Rns $argv  # 有参数 → 直接卸载
    else
        set pkg (pacman -Qq 2>/dev/null | fzf --multi --prompt "Remove> " --preview 'pacman -Qi {1} 2>/dev/null | head -30' --preview-window right:50%)
        if test -n "$pkg"
            sudo pacman -Rns $pkg  # 无参数 → fzf 搜已装包 → 卸载
        end
    end
end
用法 效果
pac 弹出 fzf 搜索所有仓库包，右侧预览包信息 → 选中后 sudo pacman -S
pac neovim 直接 sudo pacman -S neovim，不弹 fzf
pacr 弹出已安装包列表，右侧预览 → 选中后 sudo pacman -Rns
pacr firefox 直接 sudo pacman -Rns firefox，不弹 fzf

环境变量查看：envf

function envf -d "用 fzf 搜索环境变量"
    if type -q fzf
        set var (env | fzf --prompt "Env> " --preview 'echo $ {1}' | cut -d= -f1)
        if test -n "$var"
            echo "$$var = "$$var
        end
    end
end
用法 效果
envf 弹出 fzf 搜索所有环境变量 → 选中后显示变量名和值

历史命令直接执行：fh

function fh -d "用 fzf 搜索历史并执行"
    if type -q fzf
        set cmd (history | fzf --tac --prompt "History> " --preview 'echo {}' --preview-window down:3)
        if test -n "$cmd"
            eval $cmd  # 无参数 → fzf 搜历史 → 直接执行（不编辑）
        end
    end
end
用法 效果
fh 弹出 fzf 搜索历史 → 选中后直接执行（与 Atuin 的 Ctrl+R 互补：Atuin 插入命令，fh 直接执行）

tldr 覆盖 man

function man -d "优先 tldr，fallback 到系统 man"
    if count $argv >/dev/null
        if test (count $argv) -eq 1
            if tldr $argv[1] >/dev/null 2>&1
                tldr $argv[1]   # 单参数 → 先尝试 tldr 速查表
                return 0
            end
        end
        command man $argv       # 多参数或 tldr 没有 → 回退系统 man
    else
        command man
    end
end
用法 效果
man tar 先显示 tldr 速查表（常用示例），没有才显示传统 man
man 3 printf 多参数直接走系统 man，不受影响

⚙️ 配置调优建议

1. 禁用 fish_greeting（不显示 fastfetch）

function fish_greeting
    # 空函数 = 不显示任何内容

2. 修改 pay-respects 触发词为 fuck

set -gx PAY_RESPECTS_ALIAS "fuck"

3. 调整 done 通知阈值

set -U __done_min_cmd_duration 5000    # 改为 5 秒
set -U __done_notification_urgency_level normal

4. 调整 fzf 预览窗口宽度

set -gx FZF_CTRL_T_OPTS "--preview 'bat --color=always --line-range :500 {}' --preview-window right:40%"

starship timings    # 查看哪些模块耗时最长

🆘 故障排查

问题 原因 解决
Ctrl+T 没反应 fzf 未安装或未绑定 sudo pacman -S fzf && source ~/.config/fish/config.fish
f 没反应 pay-respects 未安装 paru -S pay-respects
z 命令找不到 zoxide 未初始化 检查 `zoxide init fish
yy 报错 yazi 未安装 sudo pacman -S yazi

git diff 没有颜色 delta 未配置 执行 git config 命令（见上文）
man 页面没有颜色 bat 未安装 sudo pacman -S bat
abbr 不展开 输入后没有按 Space abbr 需要按 Space 或 Enter 触发

gco 弹出 fzf 失败 之前 gco 是 abbr 删除 abbr -a -- gco 'git checkout' 这一行
v/c/k 报错 fzf 或 fd 未安装 sudo pacman -S fzf fd
